Customer Segmentation Project

Overview

This project performs customer segmentation based on various features such as age, annual income, and spending score. It utilizes the K-means clustering algorithm to group customers into distinct segments, providing insights into their behavior.

Code Explanation

The code reads customer data from a CSV file, preprocesses it, and performs customer segmentation using K-means clustering. It also generates visualizations to explore the data and visualize the clusters.

Results

The code outputs the labels assigned to each customer based on the clusters. It also displays the cluster centers for each feature combination.

Visualizations

The code generates various visualizations, including distribution plots, count plots, bar plots, and scatter plots.

Usage

  1. Download the Mall_Customers.csv file and place it in the same directory as the script.
  2. Execute the code in your preferred Python environment.

Dependencies

This project requires the following Python libraries:

  • pandas
  • numpy
  • matplotlib
  • seaborn
  • scikit-learn

Install dependencies using the command: pip install pandas numpy matplotlib seaborn scikit-learn

You can find the code in the customer_segmentation.py file in the repository.
